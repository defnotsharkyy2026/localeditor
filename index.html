🎬 Screenplay Mode (.fountain)

  • Scene Headings: Start with INT. or EXT.
  • Characters: Name in ALL CAPS on its own line.
  • Parentheticals: (inside parentheses) on its own line.

📝 Markdown (.md)

Use # for Titles, ** for Bold, and - for Lists.
